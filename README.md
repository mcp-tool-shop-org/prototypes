# Prototypes

Archived prototypes from MCP Tool Shop — consolidated into a single monorepo for reference.

These packages were previously published under `@mcptoolshop` on npm and have been deprecated or removed.

## Packages

| Package | Original npm name | Description |
|---------|-------------------|-------------|
| [mcpt](packages/mcpt) | `@mcptoolshop/mcpt` | CLI for discovering and running MCP Tool Shop tools |
| [pathway](packages/pathway) | `@mcptoolshop/pathway` | Append-only journey engine where undo never erases learning |
| [physics-svg](packages/physics-svg) | `@mcptoolshop/physics-svg` | Deterministic 2D physics engine with SVG rendering |
| [ai-music-sheets](packages/ai-music-sheets) | `@mcptoolshop/ai-music-sheets` | Piano sheet music in hybrid JSON + musical-language format |
| [websketch-demo](packages/websketch-demo) | `@mcptoolshop/websketch-demo` | Interactive demo site for WebSketch IR |
| [clearance-opinion-engine](packages/clearance-opinion-engine) | `@mcptoolshop/clearance-opinion-engine` | Deterministic name-availability and clearance-opinion engine |
| [nameops](packages/nameops) | `@mcptoolshop/nameops` | Name clearance orchestrator — batch runs, publish, and PR automation |
| [mcpt-link-fresh](packages/mcpt-link-fresh) | `@mcptoolshop/mcpt-link-fresh` | Evergreen link sync and drift fixer |
| [vector-caliper](packages/vector-caliper) | `@mcptoolshop/vector-caliper` | Scientific instrument for faithful model-state visualization |
| [mcpt-publishing-assets](packages/mcpt-publishing-assets) | `@mcptoolshop/mcpt-publishing-assets` | Logo, icon, and image asset generation for mcpt-publishing |

## Status

All packages in this repo are **archived prototypes**. They are not actively maintained and are kept here for reference only.
